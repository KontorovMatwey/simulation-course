В работе реализовано WinForms-приложение на C#, в котором обе задачи вынесены в один интерфейс и разделены на две части: слева — моделирование дискретной случайной величины, справа — моделирование нормальной случайной величины. Основная логика находится в `MainForm.cs`: там создаётся GUI, принимаются параметры, запускаются серии экспериментов и выводятся результаты. Для дискретной СВ пользователь задаёт пять **вероятностей** для исходов `1..5`. Каждое значение должно лежать в диапазоне `[0,1)`, а их сумма должна быть равна 1; если это условие нарушено, программа выдаёт ошибку. После серии испытаний вычисляются эмпирические вероятности, выборочное среднее, выборочная дисперсия и статистика хи-квадрат. Для удобства добавлены две кнопки: отдельный запуск для своего `N` и отдельный запуск для базового набора `N = 10, 100, 1000, 10000`. Теоретическая база здесь — ряд распределения дискретной случайной величины, формулы для математического ожидания, дисперсии и критерий Пирсона ??.

Во второй части моделируется нормальная случайная величина методом Бокса–Мюллера. Пользователь задаёт математическое ожидание, дисперсию и объём выборки, после чего программа генерирует выборку, строит гистограмму и накладывает теоретическую плотность нормального распределения. Далее вычисляются выборочное среднее, выборочная дисперсия, их относительные погрешности и статистика хи-квадрат для проверки согласия эмпирического распределения с теоретическим. Как и в первой части, предусмотрен отдельный запуск для одного `N` и базовый запуск для `N = 10, 100, 1000, 10000`. В коде это реализовано через отдельные методы генерации, расчёта статистик и отрисовки графиков, что делает структуру понятной и простой для защиты. Теоретическая база здесь — свойства нормального распределения, метод Бокса–Мюллера и критерий ?? для проверки гипотезы о соответствии распределению.
