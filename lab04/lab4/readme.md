В ходе работы был реализован простой конгруэнтный генератор случайных чисел и проведено сравнение его характеристик со встроенным генератором языка. Для выборки из 100000 значений выборочные среднее и дисперсия оказались близки к теоретическим значениям для равномерного распределения на отрезке [0,1): среднее около 0.5, дисперсия около 1/12. Это показывает, что оба генератора в целом подходят для получения псевдослучайных чисел, но встроенный генератор обычно даёт более стабильные результаты. Конгруэнтный генератор при этом проще по реализации, но имеет более ограниченное качество случайности.
