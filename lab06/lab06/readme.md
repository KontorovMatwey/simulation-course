В работе реализовано WinForms-приложение на C#, в котором обе задачи вынесены в один интерфейс и разделены на две части: слева - моделирование дискретной случайной величины, справа - моделирование нормальной случайной величины. Основная логика находится в `MainForm.cs`: там создаётся GUI, принимаются параметры, запускаются серии экспериментов и выводятся результаты. Для дискретной СВ пользователь задаёт пять **вероятностей**, которые внутри расчёта нормируются до суммы 1, а затем по ним генерируются значения. После серии испытаний вычисляются эмпирические вероятности, выборочное среднее, выборочная дисперсия и статистика хи-квадрат. Для удобства добавлены две кнопки: отдельный запуск для своего `N` и для базового набора `N = 10, 100, 1000, 10000`. В графике показано сравнение теоретического и эмпирического распределений. Теоретическая база здесь - ряд распределения дискретной случайной величины, формулы для математического ожидания, дисперсии и критерий Пирсона Хи-квадрат.

Во второй части моделируется нормальная случайная величина методом Бокса–Мюллера. Пользователь задаёт математическое ожидание, дисперсию и объём выборки, после чего программа генерирует выборку, строит гистограмму и накладывает теоретическую плотность нормального распределения. Далее вычисляются выборочное среднее, выборочная дисперсия, их относительные погрешности и статистика хи-квадрат для проверки согласия эмпирического распределения с теоретическим. Как и в первой части, предусмотрен отдельный запуск для одного `N` и базовый запуск для `N = 10, 100, 1000, 10000`. Теоретическая база здесь - свойства нормального распределения, метод Бокса-Мюллера и критерий Хи-квадрат для проверки гипотезы о соответствии распределению.
