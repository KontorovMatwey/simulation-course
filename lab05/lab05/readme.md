# Моделирование случайных событий (GUI)

В данном приложении реализовано моделирование случайных событий с использованием линейного конгруэнтного генератора (LCG) — частного случая конгруэнтных генераторов псевдослучайных чисел. Генератор задаётся параметрами ( M = 2^{63} ) и ( \beta = 2^{32} + 3 ), что соответствует мультипликативной форме LCG без добавочной константы. На каждом шаге вычисляется новое значение последовательности, которое затем нормируется к диапазону ([0, 1)).

Приложение содержит две части: «Скажи да или нет» и «Magic 8-Ball». В первой используется вещественное значение генератора для принятия бинарного решения с настраиваемой вероятностью, во второй - целочисленное значение для выбора одного из 16 вариантов ответа. Для каждого из вариантов во втором случае задаётся вес, который и определяет вероятность выпадения того или иного варианта
